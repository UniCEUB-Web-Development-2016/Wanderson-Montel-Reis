Projeto Analítico Laboratorial

O objetivo deste projeto é fornecer análises para diversos pacientes que serão cadastrados no sistema. Este projeto foi criado por wanderson montel dos reis, graduante do curso de ciência da computação, ao surgir a necessidade de ver de forma mais clara as informações nutricionais sobre os pacientes. Consiste em salvar os dados do usuário no banco e usá-las como fonte para o gerador de análises.
